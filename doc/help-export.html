Exporting packages

A selection of packages can be exported to a new file. The first step in exporting packages, is to create a selection. After that, open the “File” menu and select “Export”

The export file is in the format of a repository, as described in the File format specification. It can be opened and edited in the Packages application, like any other repository.

Packages exported from one repository can subse­quently be imported into another repository.
